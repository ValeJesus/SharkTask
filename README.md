# SharkTask
Frontend de um Gerenciador de Tarefas desenvolvido com HTML, CSS e JavaScript. O objetivo deste projeto é praticar estruturação de interfaces, manipulação do DOM e preparação para integração futura com uma API Java Spring Boot.  Funcionalidades:  Adicionar tarefas Marcar tarefas como concluídas Editar tarefas Excluir tarefas Interface responsiva
