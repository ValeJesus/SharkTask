# Gerenciador de Tarefas - Frontend

## Sobre o Projeto

Este projeto é o frontend de um sistema de gerenciamento de tarefas desenvolvido como parte de um plano de estudos de desenvolvimento web com Java.

O objetivo inicial é construir toda a interface utilizando HTML, CSS e JavaScript puro para compreender os fundamentos do desenvolvimento web antes da integração com o backend em Java utilizando Spring Boot.

---

## Funcionalidades

* Criar novas tarefas
* Visualizar lista de tarefas
* Editar tarefas existentes
* Excluir tarefas
* Marcar tarefas como concluídas
* Interface simples e intuitiva

---

* HTML5

* JavaScript
